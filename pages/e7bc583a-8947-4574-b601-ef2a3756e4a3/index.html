Using the rule of dominance it is possible to take two parents and predict what genotype combinations are possible for the offspring. Since it can be taxing to keep track of this characteristic or that one and keep the dominant and recessive traits straight, a simple naming convention is used to make life simpler. The dominant characteristic is represented by a capital letter like A. The recessive trait is listed as a small caps letter like a. In this way one can figure out the offspring for any given parents. There are 3 possible types of parents:

aa

Using these 3 types of parents it is possible to perform a number of mating crossing each one giving a different combination of offspring genotypes.

homozygous dominant x homozygous dominant

AA x AA

homozygous dominant x homozygous recessive

AA x aa

homozygous dominant x heterozygous

AA x Aa

homozygous recessive x heterozygous

aa x Aa

heterozygous x heterozygous

Aa x Aa

These crosses are typically represented by using a grid called a Punnett square. Using these one can see what types of combinations are produced. It is possible to calculate the probability of getting a particular result or genotype in a new offspring. An example could be used with seed pod color.
